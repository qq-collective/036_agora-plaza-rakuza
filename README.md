# 🎬 SCENARI AGORA STUDIO
> ここから飛び立つ話題を探せ

**SCENARI AGORA STUDIO** は、セナリー学院のPro相談会ナレッジを蓄積・分類・活用するための動的ナレッジベースです。  
毎週の相談会テキストをペーストするだけで、タイトルと収録日を自動解析してテーブルに追加できます。

---

## 🚀 使い方

### 1. ナレッジページのテキストをコピー
サイトのナレッジページを開き、テキスト全体をコピーします。

以下のフォーマットに対応しています：
```
2026年3月18日 Pro相談会ナレッジ
1. 友人のホームページを納品 - Claude Codeでマニュアルも自動生成
2. Playwrightで自動テスト・操作デモ動画も撮れる
3. マンション管理アプリをMDファイル駆動で開発
...
```

### 2. 左パネルに貼り付ける
アプリを開いて左側の入力エリアにペーストすると、**日付とタイトルが自動解析**されてプレビューに表示されます。

### 3. 「テーブルに追加」ボタンを押す
追加されたデータには **NEW** バッジが5秒間表示されます。

### 4. Hidden Codeを確認・修正
AIが自動分類しますが、ドロップダウンで手動修正できます。

## 🔍 フィルター・検索

- 上部のボタンで **Hidden Code別フィルター** が可能
- 右上の検索ボックスで **タイトルキーワード検索** が可能
- 収録日は **クリックで直接編集** できます（`YYYY-MM-DD` 形式）
